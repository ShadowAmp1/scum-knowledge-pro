# SCUM DB PRO v4.7

## Что обновлено

- обновлена вкладка **Бункеры**;
- добавлены карты к обычным бункерам, обновлённым пользователем;
- добавлены схемы для:
  - Z0
  - Z2
  - A2
  - B0
  - B4
  - C1
  - D3
- карты заброшенных бункеров сохранены;
- блок с картами на странице бункера теперь универсальный и подходит как для обычных, так и для заброшенных бункеров;
- обновлены описания бункеров, у которых теперь есть карты.

## Примечание

- для обычного бункера **D0** карта пока не добавлена, потому что пользователь не присылал схему именно этого бункера.
