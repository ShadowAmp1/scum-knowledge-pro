# SCUM DB PRO v4 — Интерактивная карта

## Добавлено

- Новый раздел `/map` с интерактивной тактической картой.
- Поиск по меткам: название, сектор, описание, назначение.
- Фильтры по категориям:
  - Бункеры;
  - Торговцы;
  - Лут;
  - Транспорт;
  - Места под базу;
  - Опасные зоны;
  - Города;
  - Военные зоны.
- Фильтр по уровню риска: низкий, средний, высокий, экстрим.
- Карточка выбранной точки с:
  - сектором;
  - tier лута;
  - риском;
  - описанием;
  - списком “подходит для”;
  - рекомендуемым набором;
  - быстрым переходом в связанный раздел.
- Расширенный файл данных `src/data/mapMarkers.ts`.
- Новый компонент `src/components/InteractiveMap.tsx`.
- Дополнительные стили карты в `src/app/globals.css`.
- Обновлена версия проекта до `4.0.0`.

## Важно

Карта сделана как PRO-интерактивный слой без внешних библиотек. Это значит, что проект не требует Leaflet/Mapbox и проще запускается на Render.

Координаты меток можно менять в файле:

```txt
src/data/mapMarkers.ts
```

Поля `x` и `y` — это проценты положения на карте от 0 до 100.
