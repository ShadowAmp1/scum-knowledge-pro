# SCUM DB PRO v4.6 — обновленная вкладка бункеров

- Полностью переделана вкладка `/bunkers`.
- В базе оставлены подтвержденные по скриншотам бункеры.
- Обычные бункеры: Z0, Z2, A2, B0, B4, C1, D0, D3.
- Заброшенные бункеры: A1, A3, C4, D1.
- Для заброшенных бункеров добавлены карты уровней -01 и -02.
- Добавлена галерея карт на страницах заброшенных бункеров.
- Обновлены фильтры, описание, подготовка, маршруты и советы.
