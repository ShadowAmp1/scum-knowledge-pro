# SCUM DB PRO v4.3

## Заброшенные бункеры

Добавлены первые точные метки заброшенных бункеров по присланным квадратам карты:

- A1 — юго-запад сектора, рядом с береговой дорогой.
- A3 — юго-западная часть сектора, возле поселения и белой дороги.
- C4 — северо-восточная часть сектора, возле белой дороги и малого комплекса.
- D1 — северо-восточная часть сектора, рядом с озером и желтой дорогой.

## Карта

- Добавлена отдельная категория `Заброшенные бункеры`.
- По умолчанию карта показывает обычные и заброшенные бункеры.
- Заброшенные бункеры получили фиолетовые метки с отдельным описанием.
- Сетка поверх карты остается отключенной.

## Точность

Координаты рассчитаны по положению маркера на присланных скриншотах каждого сектора и переведены в проценты относительно общей карты.
