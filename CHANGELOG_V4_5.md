# SCUM DB PRO v4.5

## Изменения

- На карте оставлены только бункеры, которые пользователь прислал скриншотами.
- Убраны остальные обычные high-tech bunker метки, которые были расставлены приблизительно.
- Сохранены уточненные обычные бункеры: Z0, Z2, A2, B0, B4, C1, D0, D3.
- Сохранены уточненные заброшенные бункеры: A1, A3, C4, D1.
- Убран эффект увеличения/сдвига метки при наведении.
- Подпись метки теперь появляется абсолютно под точкой и не меняет положение самой метки.

## Где править координаты

Файл:

```txt
src/data/mapMarkers.ts
```

Поля:

```ts
x: 50,
y: 25,
```

`x` — слева направо, `y` — сверху вниз, в процентах от всей карты.
