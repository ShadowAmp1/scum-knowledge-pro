# SCUM DB PRO v4.2 — Accurate Bunker Map

## Что изменено

- Убрана дополнительная сетка поверх изображения карты.
- По умолчанию на карте показываются только бункеры, чтобы не было визуального мусора.
- Добавлены основные обычные high-tech бункеры, которые сейчас есть в файлах данных проекта. Полный набор из 25 обычных бункеров остаётся задачей для следующих версий.
- Safe Zone / торговцы вынесены отдельной категорией.
- Метки стали меньше, подпись показывается у выбранной точки или при наведении.
- Координаты всех точек вынесены в `src/data/mapMarkers.ts`.

## Как двигать точку

В файле `src/data/mapMarkers.ts` у каждой точки есть:

```ts
x: 50.2,
y: 24.3,
```

`x` — слева направо в процентах от карты.  
`y` — сверху вниз в процентах от карты.
