# SCUM DB PRO v4.17

- внутри раздела **Оружие** добавлена новая вкладка **Обвесы**;
- добавлена страница `/weapons/attachments`;
- добавлены отдельные карточки `/weapons/attachments/[slug]`;
- внесено **84 обвеса** по категориям:
  - Обвесы на лук
  - Магазины
  - Прицелы
  - Фонарики
  - Планки
  - Глушители
- добавлены поиск и фильтры по обвесам;
- добавлена сверка совместимости с оружием из текущей базы;
- на странице оружия теперь показывается блок **Подходящие обвесы**;
- для обвесов добавлены роли, советы, где искать и примечания по совместимости;
- версия проекта обновлена до **4.17.0**.
