# SCUM DB PRO v3

## Добавлено

- Новый расширенный раздел **Бункеры**.
- Поиск по бункерам.
- Фильтры по риску, типу, tier лута и удобству для новичка.
- Расширенные страницы каждого бункера: подготовка, оружие, лут, угрозы, маршрут, ошибки, советы соло и дуо.
- Новый улучшенный раздел **Лут**.
- Поиск и фильтры лута по категории, редкости и приоритету.
- Новый раздел **Подготовка к рейду**.
- Готовые raid-kit чек-листы: стартовый, duo и PRO.
- Обновлено меню и карточки главной страницы.

## Как обновить на GitHub

```bash
git add .
git commit -m "Add SCUM DB PRO v3 bunkers loot preparation"
git push
```

Render после push должен автоматически пересобрать сайт. Если нет — нажми Manual Deploy → Deploy latest commit.
