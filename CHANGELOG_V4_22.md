# SCUM Knowledge PRO v4.22-lite

Добавлено:
- глобальный поиск по всему сайту: оружие, обвесы, лут, бункеры, карта, гайды, транспорт и подготовка;
- страница сравнения оружия `/weapons/compare` с выбором 2–3 вариантов, таблицей рейтингов, плюсами/минусами и подходящими обвесами;
- Admin Lite `/admin` — локальная демо-админка для подготовки JSON-черновиков оружия, лута, маркеров карты и гайдов без базы данных;
- ссылки в меню, главных разделах и sitemap;
- валидация внутренних ссылок обновлена под новые страницы.

Важно: Admin Lite не редактирует файлы на сервере Render. Для полноценной v5-админки нужны авторизация, API и база данных.
