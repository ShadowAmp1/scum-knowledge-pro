## 🔧 Установка и запуск

### 1. Установка Node.js
```bash
# Windows (скачать с https://nodejs.org)
# Установить LTS версию (20.x или выше)

# Проверка установки
node --version
npm --version
```

### 2. Установка зависимостей
```bash
cd scum-knowledge-pro
npm install
```

### 3. Исправление TypeScript ошибок
```bash
# Добавить свойство difficulty в тип WeaponRating
# В файле src/data/weapons.ts добавить:
difficulty: number;
```

### 4. Запуск локально
```bash
npm run dev
# http://localhost:3000
```

### 5. Сборка для production
```bash
npm run build
npm start
```

## 🏗️ Развертывание на хостинге

### Vercel (рекомендуется)
1. Подключить репозиторий к Vercel
2. Настройки сборки:
   - Build Command: `npm run build`
   - Output Directory: `.next`
   - Node Version: `20.x`

### Docker
```dockerfile
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "start"]
```

### Railway/Render
1. Подключить репозиторий
2. Установить переменные окружения:
   - NODE_ENV=production
   - Node Version: 20
